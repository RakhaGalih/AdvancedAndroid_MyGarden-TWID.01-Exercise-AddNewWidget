### pengertian widget
Widget adalah sebuah “perwakilan” dari aplikasi yang memungkinkan pengguna untuk dapat
menggunakan fitur dari aplikasi yang diwakilkan tersebut secara langsung tanpa harus membuka aplikasi “sesungguhnya”
